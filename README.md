## 📌 Overview

**DD RSS Reader** is an Enigma2 plugin that allows you to read RSS feeds directly on your receiver.

Based on the work from **RSS Simple by DDamir v0.2**.

Adapted for **Python 3** and **Full HD screens** by **@lululla (2024-05-24)**.

---

## ✨ Features

* 📡 RSS feed reader for Enigma2
* 🖥 Optimized for Full HD (FHD) screens
* ⚡ Lightweight and fast
* 🐍 Python 3 compatible
* 🎛 Simple and intuitive interface

---

## 📦 Installation

1. Copy the plugin folder to:

   ```
   /usr/lib/enigma2/python/Plugins/Extensions/
   ```

2. Restart Enigma2 (GUI restart)

---

## 🚀 Usage

* Open the plugin from the Extensions menu
* Add or manage your RSS feeds
* Browse and read news directly on your device
